This documentation is intended to present the outputs of a sample dataset.

Launch the workflow to find the motifs

The workflow consists of 3 steps described in the main package documentation. These 3 steps will be applied on our sample.

Plot the result

The function runVisualization allows to launch an interactive interface to visualize the result. There is 2 ways to visualize the result.

  • Plot the intensity of values and highlight one motifs or the top 5
# Plot the intensity of the dataset and highlight one selected motifs
intensityDataset(dataset = STMotif::example_dataset,rankList = ranksttightmotifs,rank = 1,alpha = 7)

  • Plot the spatial-time series by select a range of columns in the dataset and highlight one motifs
# Plot of 5 specific spatial series which some of them contain the best motif.
displayPlotSeries(dataset = STMotif::example_dataset, rankList = ranksttightmotifs ,rank = 1 ,space = c(1,2,5:7))
